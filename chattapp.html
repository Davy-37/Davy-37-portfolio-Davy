ChattApp : Application de Chat avec Django

Description : ChattApp est une application web interactive permettant la communication en temps réel entre utilisateurs. Le backend est développé en Django, avec une interface utilisateur en HTML, CSS, et JavaScript.

Fonctionnalités principales :

  • Inscription & Connexion sécurisée
  • Envoi/Réception de messages en temps réel
  • Création de salons de discussion
  • Responsive design

Technologies utilisées :

  • Django (Python)
  • HTML/CSS/JavaScript
  • WebSockets (Django Channels)
  • SQLite/PostgreSQL
